# sqlserver-csv-iso19110-geonetwork
Query to generate an overview of columns that can be used as CSV input for ISO19110 metadata in GeoNetwork

### Geographical data in SQL Server database with SDE
The query is written for a SQL Server database with SDE.

### CSV input ISO19110 metadata GeoNetwork
When editing ISO19110 metadata in GeoNetwork it is possible to add columns in a CSV format. The output of the query can be copy-pasted in the input field and the columns will be added to the metadata.
